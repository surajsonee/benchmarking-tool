these 2 packages are used to predict the amount of kwh and gj a house will use in a given month. 

appliance_predictions 
    relies on home predictions 
    
    energy star file has to be in the directory where the function will be called see test.py for details 

home_prediction 
    makes a home model, assumptions and calculations all from a bill_estimate function call 

    for the bill_estimation function to work the home model only need a squarefootage value and year build value for the house 

these packages require the installation of 
    numpy
    pandas
    copy
